# SEO Action Plan - Jordan Licon Photography Home Page

## Immediate Quick Wins

1. Shorten the title tag.
   - Current: `Jordan Licon - El Paso Commercial Photographer & Cinematographer`
   - Suggested: `El Paso Commercial Photographer | Jordan Licon`

2. Shorten the meta description.
   - Current length: `176` characters
   - Suggested: `El Paso commercial photographer and cinematographer for executive headshots, architecture, editorial campaigns, drone work, and brand production.`

3. Strengthen the hero keyword signal.
   - Keep the brand line `Cinematic light. Editorial eye.`
   - Add a visible supporting line near the H1: `El Paso commercial photographer and cinematographer for headshots, architecture, editorial, and brand campaigns.`

4. Add BreadcrumbList schema.
   - Home page breadcrumb can be simple: Home.
   - Service pages should use Home → Service.

## High-Impact Improvements

5. Convert key CSS background images to semantic images.
   - Priority sections: Talent Portraiture, Architecture preview, Commercial Clients if visual cards are added.
   - Benefit: better image SEO, accessibility, and clearer image indexing.

6. Replace remote Facebook video posters.
   - Save local poster images under `assets/`.
   - Update video `poster` attributes to local files.
   - Benefit: reliability and performance.

7. Add service-specific schema.
   - Headshots: `Service`
   - Architecture photography: `Service`
   - Commercial photography: `Service`
   - Drone / aerial production: `Service`
   - Connect each to the main `ProfessionalService` entity.

8. Improve readability in dense sections.
   - Use shorter sentences in About, Institutions, and Services.
   - Add short scannable intros before long proof lists.
   - Keep the premium tone, but reduce sentence length.

## Strategic SEO Work

9. Publish and verify technical files.
   - `robots.txt`
   - `sitemap.xml`
   - `llms.txt`
   - Submit sitemap to Google Search Console.

10. Run a live PageSpeed audit after deployment.
   - Check mobile LCP, INP, CLS.
   - Pay special attention to hero image, Google Fonts, remote Wix images, and videos.

11. Build internal topical clusters.
   - Headshot cluster: executive, actor, pageantry, corporate team.
   - Architecture cluster: architects, interiors, hospitality, civic, commercial real estate.
   - Drone cluster: architecture, commercial production, hospitality, land/development.
   - Commercial cluster: brand campaigns, editorial production, institutions, agencies.

12. Add stronger local trust proof.
   - Link to Google reviews if available.
   - Keep NAP consistent: `2201 East Mills Ave, 2nd Floor, El Paso, TX 79901`.
   - Match all structured data to the live Google Business Profile.

## Verification Checklist

- Title under roughly `60` characters.
- Meta description around `150-160` characters.
- One H1 with clear service/location intent.
- All important images have alt text and dimensions.
- No placeholder links.
- No missing local assets.
- JSON-LD validates in Google Rich Results Test / Schema validator.
- Mobile PageSpeed checked after publishing.
